Step 1 
In VScode terminal,
Navigate to the folder with name "AWSLabs"
Run command git clone https://github.com/awscloudessentials/intro-to-aws-lab.git (this will clone this repo into your local machine)
Run command cd intro-to-aws-lab
Run command ls -la to view content of local repo
Copy Lab3a.txt into the folder "AWSLabs"
Delete the cloned repo

Step2  (Create a Virtual Server)
Login into your AWS Console 
Create a Launch template 
Select your default VPC
With the following  Inbound rule
Type      Source
SSH         Anywhere
HTTP       Anywhere IPv4
Select “Amazon Linux 2023 AMI “ under Application and OS Images
Select “t3micro “ for instance type
Key pair with name "Linux" and key file format ".pem"
Download pem file into your local machine
Create a Security Group with name “PublicSG” and give it any description you prefer
Scroll down to the section with userdata copy and paste this:

Step3 
Create an Auto Scaling Group(ASG) using the launch template you created in Step2
Select no scaling policy and use Minimum -1
Desired - 2
Maximum - 3
Once done, move to step 4

Step 4
Go to your EC2 dashboard, wait for the 2 instances to come up and then terminate them
Send a screenshot showing the instances terminating and 2 new instances spun up by the ASG and share on WhatsApp
Once confirmed, delete your ASG
